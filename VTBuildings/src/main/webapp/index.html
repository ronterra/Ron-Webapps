VT Buildings API

Description

This application uses the REpresentational State Transfer (REST) software architectural style to provide RESTful Web Services as an API. It employs the Java platform Jakarta RESTful Web Services. It returns data in the JavaScript Object Notation (JSON) format from the VTBuildingsDB relational database.

The created API for RESTful Web Services enables a software application to obtain data from the VTBuildingsDB relational database in JSON format by using URLs under a predefined set of queries over the Internet. Each of the following URLs is called an API query, API call or API RESTful web service. JSON file returned from the API is intended for use only by a software application. APIs, JSON files, or RESTful Web Services are created for use by software applications; not by human beings.
